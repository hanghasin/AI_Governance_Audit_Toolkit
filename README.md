# 🛡️ AI Governance & Bias Audit Toolkit
**An end-to-end framework for detecting and mitigating algorithmic bias in dynamic pricing systems.**

---

## 🚀 Project Overview
This project simulates a real-world dynamic pricing engine (inspired by platforms like Booking.com) and provides a professional-grade audit and mitigation pipeline. It identifies and resolves **"Proxy Bias"** where device types (Mac/iPhone) are used as proxies for socio-economic status to apply unfair price premiums.

## 🛠️ Technical Stack
- **Language:** Python 3.12.9
- **Database:** SQLite (SQLTools integration)
- **Fairness Framework:** [Fairlearn](https://fairlearn.org/) (MetricFrame & CorrelationRemover)
- **Data Science:** Pandas, Scikit-learn
- **CI/CD:** GitHub Actions (Automated Audit Pipeline)

## 📊 The Audit Journey

### Phase 1: Bias Detection
Using **Disparate Impact Analysis**, I identified a systemic bias in the pricing algorithm:
- **Mac/iPhone Selection Rate (High Price):** ~48%
- **Android/Windows Selection Rate (High Price):** ~37%
- **Initial Bias Ratio:** **1.30** (Non-Compliant per EU AI Act standards)

### Phase 2: Algorithmic Mitigation
Applied **CorrelationRemover** (Pre-processing) to orthogonalize the relationship between `device_type` and `final_price`.



### Phase 3: Results
| Metric | Pre-Mitigation | Post-Mitigation | Status |
| :--- | :--- | :--- | :--- |
| Bias Ratio | 1.30 | **1.11** | ✅ Compliant |
| Mac Premium | +15% | **Removed** | ✅ Fixed |

## ⚖️ Governance Statement (The CEO Perspective)
As the project lead, I made a strategic decision to **prioritize ethical compliance over short-term revenue**. 
- **The Trade-off:** Mitigating the 1.30 bias ratio resulted in a ~10% reduction in premium segment revenue but successfully eliminated the legal risk of a **4% global turnover fine** under the EU AI Act.
- **The Philosophy:** Systematic fairness is a long-term brand asset that outweighs the "shadow profit" generated by discriminatory algorithms.

2. Run audit: `python3 fair_remedy.py`
3. Run mitigation: `python3 fair_mitigation.py`
